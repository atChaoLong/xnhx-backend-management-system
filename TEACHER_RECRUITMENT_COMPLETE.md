## 🚀 使用方式

### 1. 新建面试
```
仪表盘 → 老师面试 → "新增面试"
  ↓
填写基本信息（必填: 姓名、微信号、简历）
上传形象照（可选）
选择岗位需求
填写约面信息
  ↓
保存 → 创建候选人记录
```

### 2. 编辑并完善信息
```
老师面试列表 → 点击"编辑"
  ↓
逐个完成6个Tab的信息
  ├─ Tab 1: 基本信息 (已有，可补充)
  ├─ Tab 2: 约面信息 (已有，可补充)
  ├─ Tab 3: 面试评分 (新增)
  ├─ Tab 4: 素质评价 (新增)
  ├─ Tab 5: 复核流程 (改进)
  └─ Tab 6: 谈薪入库 (改进)
  ↓
保存 → 自动更新候选人状态
```

### 3. 复核流程
```
复核官进入编辑页面
  ↓
[Tab 5] 复核流程
  - 查看面试录像
  - 填写复核结果和评价
  - 选择: "已复核" 或 "不符合"
  - 试讲视频上传（可选）
  ↓
保存 → 状态自动变为"待入库"或"拒绝"
```

### 4. 谈薪和入库
```
HR/管理员进入编辑页面
  ↓
[Tab 6] 谈薪入库
  - 填写薪资信息
  - 选择老师级别
  - 确认毕业班资质
  - 勾选"是否入库"
  ↓
保存 → 显示"入库"按钮
  ↓
点击"入库" → 输入ClassIn密码 → 完成!
```
